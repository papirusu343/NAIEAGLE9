# NovelAI Image Generator V4.5

NovelAI V4.5対応の画像生成Webアプリケーションです。

## 機能

- **NovelAI V4.5対応**: 最新のモデルとパラメーターに完全対応
- **ワイルドカード機能**: __category__形式でランダムなプロンプト生成
- **マルチキャラクター**: 複数キャラクターの座標指定
- **Eagle連携**: 生成画像の自動保存と管理
- **詳細パラメーター**: CFG Scale、CFG Rescale、サンプラーなど
- **ダークモードUI**: レスポンシブデザイン対応
